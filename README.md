# Projeto-60-Dias-com-Logica
🚀 Projeto 60 Dias com Lógica de Programação:
Este repositório é o meu diário de bordo oficial. Aqui, eu documento a realidade nua e crua de quem decidiu construir uma base de lógica do absoluto zero. 
Sem métodos milagrosos ou atalhos que não existem. É tijolo por tijolo, para melhorar o pensamento logico usando C#.

## 🛠️ O que estou construindo e praticando?
Para aprender a pensar como um programador de verdade e entender como resolver problemas reais de trás para frente, estou aplicando a lógica pura diretamente no ecossistema C# através de:

* **Lógica de Decisão & Loops:** Criando algoritmos estruturados com condicionais (`if/else`, `switch`) e laços de repetição (`for`, `while`) para controle de fluxos complexos.
* **Manipulação de Dados & Coleções:** Praticando com arrays, matrizes e listas na memória para entender como organizar informações de forma eficiente.
* **Funções & Modularização:** Quebrando grandes problemas complexos em pequenos blocos de código reaproveitáveis.
* **Desafios Reais do Dia a Dia:** Implementando algoritmos práticos com C# (filtros de decisão, sequenciadores e simuladores) para fixar a lógica na memória de longo prazo.
